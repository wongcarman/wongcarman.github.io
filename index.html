Introduction of Crytography

RSA

Rivest–Shamir–Adleman (RSA) is the most wide-used encryption algorithm for public key cryptography. It can be used everywhere, which makes RSA as a safe default choice. RSA is widely used in SSL certificates that power secured protocols (like HTTPS, POPS, SMTPS). It is also commonly used in e-commerce protocols. Besides, it also happens to be one of the main methods used in PGP and GPG for encrypting and signing e-mails. Thus, it is important to know RSA.
